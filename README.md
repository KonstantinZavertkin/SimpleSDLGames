SnakeGame
==================================
## Installation instructions
Requirements:  
 - SDL2

Building the executable file can be done using the following commands:  
&emsp;mkdir build  
&emsp;cd build  
&emsp;cmake ..  
&emsp;make all  
  
Run:  
&emsp;./GameRunner  
  
Tested on Debian 10 and Windows 10.
